# Gestión Info

Sistema CRUD en consola para la gestión de usuarios, con persistencia de datos en formato JSON.

## Características

- Crear, listar, actualizar y eliminar usuarios.
- Persistencia de datos en `data/records.json`.
- Validaciones para `id`, `nombre` y `email`.
- Visualización de datos en formato tabular usando `pandas`.
- Menú interactivo en consola.
- Pruebas unitarias básicas incluidas.

## Requisitos

- Python 3.10 o superior
- Dependencias listadas en `requirements.txt`
